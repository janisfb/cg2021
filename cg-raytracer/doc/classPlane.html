Detailed Description

This class implements a simple plane object. A plane is specified by a center point and a normal vector. This class overrides the intersection method Object::intersect().

Compute the intersection of the plane with _ray. Return whether there is an intersection. If there is one, return the intersection data. This function overrides Object::intersect().
